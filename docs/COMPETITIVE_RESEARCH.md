## Key Insights

### What Makes Users Return
1. Meaningful notifications (endorsements, views, opportunities)
2. Content cadence (weekly digest, monthly themes)
3. Peer activity visibility
4. Opportunity matching (grants, commissions, collaborations)

### Profile Completion Benchmarks
- Sign-up → basic profile: 70-80%
- Basic → complete (3+ sections): 25-35%
- Complete → active (returns in 30 days): 15-20%

### Improvement Levers
- Completion meter: +15-20% completion
- Gated features ("complete profile to appear in search"): +10-15%
- Email drip sequence (days 1, 3, 7, 14): +10-15%
- Social proof in onboarding: +5-10%

### What NOT to Do
- Don't build algorithmic infinite-scroll feeds (burns out artists)
- Don't require design decisions for basic profiles (kills completion)
- Don't gate too many features behind completion (feels punitive)
- Don't over-engineer search before you have data volume
