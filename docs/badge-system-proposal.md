# Resonance Network — Badge System Proposal

## How It Works

Badges are small circular icons displayed next to the user's name on their profile page. When you hover over a badge (or tap on mobile), a tooltip shows the badge name, description, and date it was earned.

Badges can be awarded automatically by the system (based on activity) or manually by an admin. They are stored in a database table linked to each user's profile.


## Founding & Status Badges (admin-awarded, rare)

- Founder — Lightning bolt symbol — First 10 members to join the platform
- Featured — Diamond symbol — Profile was featured on the homepage


## Project Badges (auto-awarded when linked to a project)

- Project Collaborator — Interlocking circles symbol — Active collaborator on a specific project
- Project Lead — Star symbol — Leading or created an approved project
- Multi-Project — Double star symbol — Collaborated on 3 or more projects


## Engagement Badges (auto-awarded by activity)

- Pioneer — Sparkle symbol — Completed profile within first week of joining
- Connector — Arrows symbol — Sent 5 or more collaboration interest requests
- Portfolio Pro — Grid symbol — Added 5 or more gallery items to profile
- Networked — Concentric circles symbol — Followed 10 or more projects


## Skill Badges (auto-awarded when profile is rich)

- Builder — Hammer symbol — Listed fabrication or engineering skills
- Curator — Frame symbol — Organized or curated projects


## Visual Design

Badges appear in a horizontal row below the user's name and title on their profile. They are 20-24px circular icons in Resonance teal color. On hover, a dark tooltip appears with the badge name, description, and date earned. On mobile, users tap to see the tooltip. If a user has more than 8 badges, the extras show as "+3 more" with an expandable list.

## Admin Controls

The admin dashboard gets a Badges section where you can award any badge to any user manually, see all badges across the platform, and create custom one-off badges for special occasions.
